## Project Folder

- Main app: [exam-prep-ai](exam-prep-ai)
- Detailed docs: [exam-prep-ai/README.md](exam-prep-ai/README.md)

## Quick Start

1. Open the project folder: exam-prep-ai
2. Configure environment variables in .env.local
3. Run:
   - npm install
   - npx prisma generate
   - npx prisma db push
   - npm run dev
